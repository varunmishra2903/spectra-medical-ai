\# S.P.E.C.T.R.A – System for Precision Extraction, Classification, Triage, \& Radiomics Analysis



\## Short Description

Desktop medical AI tool with:

\- Zone 1: Gatekeeper model for triage \& routing

\- Zone 2: Specialist models (Brain, Chest, Bone)

\- Zone 3: Unified reporting with radiomics and PDF export
